Databricks documentation provides how-to guidance and reference information for data analysts, data scientists, and data engineers solving problems in analytics and AI. The Databricks Data Intelligence Platform enables data teams to collaborate on data stored in the Lakehouse. See [What is Databricks?](/aws/en/introduction/)

tip

Databricks technical documentation is organized by cloud provider. Use the cloud switcher in the upper right corner of the page to choose between Amazon Web Services, [Google Cloud Platform](https://docs.gcp.databricks.com/index.html), or [Microsoft Azure](https://learn.microsoft.com/azure/databricks/). For ADI Services, see the [ADI Services documentation](https://docs.databricks.com/adi).
