## Create message comment Public preview

`POST

/api/2.0/genie/spaces/{space_id}/conversations/{conversation_id}/messages/{message_id}/comments`

Create a comment on a conversation message.

API scopes:[`genie`](/api/workspace/api/scopes#genie)

### Path parameters

[`space_id`](/api/workspace/genie/createmessagecomment#space_id)uuidRequired

Example `"e1ef34712a29169db030324fd0e1df5f"`

The ID associated with the Genie space.

[`conversation_id`](/api/workspace/genie/createmessagecomment#conversation_id)uuidRequired

Example `"e1ef34712a29169db030324fd0e1df5f"`

The ID associated with the conversation.

[`message_id`](/api/workspace/genie/createmessagecomment#message_id)uuidRequired

Example `"e1ef34712a29169db030324fd0e1df5f"`

The ID associated with the message.

### Request body

[`content`](/api/workspace/genie/createmessagecomment#content)string

<= 5000 characters

Required

Example `"This query could be optimized by adding an index on the date column."`

Comment text content.
