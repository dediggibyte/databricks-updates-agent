note

As of February 2026, new versions of the ODBC driver are being released by Databricks and the driver has been renamed to Databricks ODBC Driver. To migrate from the legacy Simba Spark ODBC driver, see the [migration guide](/aws/en/integrations/odbc/migration).

Review the [JDBC ODBC driver license](https://databricks.com/jdbc-odbc-driver-license) before you install the driver.

Some tools require separate ODBC driver installation (like Tableau Desktop), while others include the driver (like recent Power BI Desktop releases). If installation isn't required, find more information in  [Additional resources](#additional-resources).

## Databricks ODBC Driver

To download and install the Databricks ODBC Driver:

1. Go to the [Databricks ODBC Driver download page](https://www.databricks.com/spark/odbc-drivers-download).
2. Select your operating system.
3. Download and run the installer.

The driver installs to:

* Windows 64-bit: `C:\Program Files\Databricks ODBC Driver`
* Windows 32-bit: `C:\Program Files (x86)\Databricks ODBC Driver`
* macOS: `/Library/databricks/databricksodbc`
* Linux: `/opt/databricks/databricksodbc`

## Simba Spark ODBC Driver (legacy)

important

The Simba Spark ODBC Driver is [deprecated](/aws/en/release-notes/runtime/databricks-runtime-ver#platform-feature-lifecycle). It is still available and usable, but Databricks recommends using the Databricks ODBC Driver instead.

To download the legacy Simba Spark ODBC Driver, go to the [ODBC Drivers archive](https://www.databricks.com/spark/odbc-drivers-archive). Before downloading the driver, you must sign in to your Databricks account.

If you use Databricks on AWS GovCloud, you must contact your account team to receive access to the driver.

* Windows
* MacOS
* Linux

1. Go to [All ODBC Driver Versions - Windows](https://www.databricks.com/spark/odbc-drivers-archive#windows).
2. Choose 32-bit or 64-bit for your system architecture.
3. Extract the downloaded `.zip` file.
4. Run the `.msi` installer.

The driver installs to `C:\Program Files\Simba Spark ODBC Driver`.

1. Go to [All ODBC Driver Versions - Mac OS](https://www.databricks.com/spark/odbc-drivers-archive#mac).
2. Choose 32-bit for the latest version.
3. Extract the downloaded `.zip` file.
4. Run the `.dmg` installer.

The driver installs to `/Library/simba/spark`.

1. Go to the download page for your package manager:

   * [rpm packages](https://www.databricks.com/spark/odbc-drivers-archive#linux)
   * [deb packages](https://www.databricks.com/spark/odbc-drivers-archive#linux-deb)
2. Choose 32-bit or 64-bit for your system architecture.
3. Extract the downloaded `.zip` file.
4. Install the package:

   **RPM:**

   Bash

   ```
   sudo yum --nogpgcheck localinstall simbaspark_<version>.rpm
   ```

   **DEB:**

   Bash

   ```
   sudo dpkg -i simbaspark_<version>.deb
   ```

The driver installs to `/opt/simba/spark`.
