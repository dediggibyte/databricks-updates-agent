The Account SCIM v2.1 API allows you to create and manage users, groups, and service principals in the Databricks account.

You can download a [PDF of the Account SCIM v2.1 API reference](/aws/en/assets/files/account-scim-2_1-da24f7cb52abfee4646aff48adb4307f.pdf).

## Limits

SCIM filter expressions are limited to 200 nodes. Requests with filters that exceed this limit return an `INVALID_FILTER` error with HTTP status 400. To stay within the limit, break large filters into multiple smaller requests.

For all Account SCIM API rate limits, see [Resource limits](/aws/en/resources/limits).
