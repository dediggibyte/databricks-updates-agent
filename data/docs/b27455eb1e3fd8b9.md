The Databricks connector for Microsoft Power Platform enables you to build low-code applications, workflows, and agents using your Databricks data. Build apps in Power Apps, create automated flows in Power Automate, and develop AI agents in Copilot Studio while preserving your Databricks governance controls.

## Get started

* + [Create a Databricks connection](/aws/en/integrations/msft-power-platform/setup)
  + Set up the Databricks connector on Microsoft Power Platform by creating a connection.
* + [Use Databricks data on Power Platform](/aws/en/integrations/msft-power-platform/usage)
  + Build Power canvas apps, Power Automate flows, and Copilot Studio agents using your Databricks data and Genie Agents.
