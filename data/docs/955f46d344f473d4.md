Beta

This feature is in [Beta](/aws/en/release-notes/release-types). Workspace admins can control access to this feature from the **Previews** page. See [Manage Databricks previews](/aws/en/admin/workspace-settings/manage-previews).

The managed Google Ads connector in Lakeflow Connect allows you to ingest data from Google Ads into Databricks.

## Feature availability

| Feature | Availability |
| --- | --- |
| UI-based pipeline authoring | Not supported |
| API-based pipeline authoring | Supported |
| Declarative Automation Bundles | Supported |
| Incremental ingestion | Partially supported  Report tables support incremental ingestion. Resource tables are fully refreshed on each update. |
| Unity Catalog governance | Supported |
| Lakeflow Jobs | Supported |
| SCD type 2 | Not supported |
| Column selection and deselection | Supported  You can select specific tables. |
| API-based row filtering | Not supported |
| Automated schema evolution: New and deleted columns | Not supported |
| Automated schema evolution: Data type changes | Not supported |
| Automated schema evolution: Column renames | Not supported |
| Automated schema evolution: New tables | N/A |
| Maximum number of tables per pipeline | 250 |

## Authentication methods

| Authentication method | Availability |
| --- | --- |
| OAuth U2M | Supported |
| OAuth M2M | Not supported |
| OAuth (manual refresh token) | Not supported |
| Basic authentication (username/password) | Not supported |
| Basic authentication (API key) | Not supported |
| Basic authentication (service account JSON key) | Not supported |

## Start ingesting from Google Ads

The following table provides an overview of the end-to-end Google Ads ingestion flow, based on user type:

| User | Steps |
| --- | --- |
| Admin | 1. Configure Google Ads to enable authentication from Databricks. See [Configure OAuth for Google Ads ingestion](/aws/en/ingestion/lakeflow-connect/google-ads-source-setup). 2. Use Catalog Explorer to create a connection to Google Ads so that non-admins can create pipelines. See [Create a Google Ads connection](/aws/en/ingestion/lakeflow-connect/google-ads-connection). |
| Non-admin | Use any supported interface to create a pipeline from an existing connection. See [Ingest data from Google Ads](/aws/en/ingestion/lakeflow-connect/google-ads-pipeline). |
