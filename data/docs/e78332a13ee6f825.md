Beta

This feature is in [Beta](/aws/en/release-notes/release-types). Workspace admins can control access to this feature from the **Previews** page. See [Manage Databricks previews](/aws/en/admin/workspace-settings/manage-previews).

The managed Reddit Ads connector in Lakeflow Connect allows you to ingest data from Reddit Ads into Databricks.

## Start ingesting from Reddit Ads

The following steps provide an overview of the end-to-end Reddit Ads ingestion flow:

1. **[Configure Reddit Ads for ingestion](/aws/en/ingestion/lakeflow-connect/reddit-ads-source-setup)** (Admins) — Set up Reddit Ads to authenticate with Databricks.
2. **[Create a Unity Catalog connection](/aws/en/ingestion/lakeflow-connect/reddit-ads-connection)** (Admins) — Create a connection in Catalog Explorer so non-admins can create pipelines.
3. **[Create an ingestion pipeline](/aws/en/ingestion/lakeflow-connect/reddit-ads-pipeline)** (Admins or non-admins) — Select any supported interface and create a pipeline from an existing connection.
