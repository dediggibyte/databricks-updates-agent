Genie Code is the AI coding and data assistant for developers and technical practitioners in the Databricks workspace. It generates and runs code, builds pipelines and AI/BI dashboards, debugs errors, and works directly with Unity Catalog tables, columns, and lineage to accelerate multi-step data work. It runs in notebooks, the SQL editor, the Lakeflow Pipelines Editor, AI/BI dashboards, and MLflow, and chat threads persist as you navigate between pages.

Genie Code is governed by your Unity Catalog permissions, so it can only access data and perform operations that you have permissions for. Fair usage limits prevent abuse but do not affect most users.

Genie Code is part of the Genie family of AI experiences. Business users who want a simplified chat interface to ask data questions, explore dashboards, and run apps should use [Genie One](/aws/en/genie-one/). Data teams configuring trusted data, metrics, and business rules for domain-specific environments should see [Genie Agents](/aws/en/genie/). For privacy, trust, and security details, see [Databricks AI assistive features trust and safety](/aws/en/databricks-ai/databricks-ai-trust).

## Genie Code capabilities by product surface

Genie Code adapts to the product surface you are using and handles multi-step tasks autonomously. In the Lakeflow Pipelines Editor, it focuses on pipeline editing and data engineering. In notebooks and the SQL Editor, it supports data exploration and analysis. In dashboards, it supports data analysis and dashboard creation. In MLflow, it helps debug and improve GenAI applications.

* + [Data science and ML in notebooks](/aws/en/notebooks/ds-agent)
  + Genie Code is integrated with Databricks notebooks and optimized to automate exploratory data analysis and model training.
* + [Pipeline development in Lakeflow](/aws/en/ldp/de-agent)
  + Genie Code has data engineering capabilities that make it easy to automate ETL workloads and build Lakeflow Spark Declarative Pipelines.
* + [Data analysis in dashboards](/aws/en/dashboards/manage/dashboard-agent)
  + Genie Code can plan and generate production-ready dashboards and help with data analysis.
* + [Observability & evaluation in MLflow](/aws/en/mlflow3/genai/getting-started/genie-code)
  + Genie Code can understand, debug, and improve your GenAI applications.

## Built-in features for code assistance

Genie Code uses Unity Catalog metadata (tables, columns, descriptions, and your company's popular data assets) to give personalized responses while you write, debug, and run code. Use the following built-in features:

* + [Chat with Genie Code](/aws/en/genie-code/use-genie-code#pane)
  + Ask Genie Code questions and it responds with relevant information, including citations from Databricks documentation.
* + [Inline suggestions and autocomplete](/aws/en/notebooks/code-assistant#get-inline-code-suggestions-python-and-sql-examples)
  + Get help writing code with inline suggestions.
* + [Quick fix](/aws/en/notebooks/code-assistant#quick-fix)
  + Genie Code automatically suggests fixes for basic errors that you can choose to accept and run.
* + [Diagnose error](/aws/en/notebooks/code-assistant#debug)
  + Genie Code analyzes more complex errors, including [environment errors](/aws/en/notebooks/code-assistant#debug-env), and attempts to fix them.
* + [/slash commands](/aws/en/notebooks/code-assistant#use-slash-commands-for-prompts)
  + Use /slash commands for common prompts.
* + [Command shortcuts](/aws/en/notebooks/code-assistant#notebook-assistant)
  + Work with Genie Code directly in a notebook cell.
* + [Filter data](/aws/en/notebooks/code-assistant#text2filter)
  + Filter data using natural language in supported surfaces.
* + [Explore sample data](/aws/en/discover/database-objects#natural-language)
  + Explore sample table data in Catalog Explorer using natural language in Catalog Explorer.
