The *Databricks Connector for Google Sheets* connects your Databricks workspace with Google Sheets. Use the connector to query Databricks data from within Google Sheets, enabling further analysis.

## Get started

| Feature | Description |
| --- | --- |
| [Set up the connector](/aws/en/integrations/google-sheets/connect) | Set up the Databricks Connector for Google Sheets by connecting to your Databricks workspace. |
| [Query data](/aws/en/integrations/google-sheets/query-data) | Import data into Google Sheets by selecting tables, writing SQL queries, adding query parameters, and creating pivot tables. |
| [Schedule data refreshes](/aws/en/integrations/google-sheets/schedule-refresh) | Set up automatic scheduled refreshes to keep your imported data up to date on an hourly, daily, or weekly basis. |
