Preview

This feature is in [Public Preview](/aws/en/release-notes/release-types).

The Databricks Genie mobile app lets business users access key Genie One capabilities from iOS and Android devices. Users can ask data questions in natural language, view and interact with dashboards, and run Databricks Apps without being at a desktop.

The mobile app works alongside the Genie One web experience and supports all workspace user entitlements, including [consumer access](/aws/en/ai-bi/consumers/).

With the Genie mobile app, users can:

* **Ask questions with Genie**: Answer questions with natural language by using relevant Genie Agents and Unity Catalog tables.
* **View dashboards**: View, interact with, and share existing AI/BI dashboards, optimized for mobile.
* **Chat with dashboards**: Ask follow-up questions and do root-cause analysis directly within a dashboard using Ask Genie.
* **Run Databricks Apps**: Use custom-built Databricks Apps that combine analytics, AI, and workflows. Apps built with [Databricks AppKit](https://developers.databricks.com/docs/appkit/v0/) are optimized for mobile.

For IT and security teams, see [Administer the Genie mobile app](/aws/en/genie-one/mobile-admin) for configuration guidance, network security requirements, authentication setup, and Mobile Device Management (MDM) deployment.

## Supported devices

* iOS on iPhone and iPad
* Android on all device sizes

## Enable the app

The Genie mobile app is enabled by default for all accounts in Public Preview. If it is not already enabled, an account admin can enable it manually:

1. In the account console, go to **Previews** in the left nav.
2. Search for **Genie One Mobile**.
3. Enable **Genie One Mobile**.

## Install the Genie mobile app

Download the Genie app from the Apple App Store (iOS) or Google Play Store (Android).

* IOS
* Android

1. On your iPhone or iPad, open the App Store.
2. Search for **Databricks Genie**.
3. Download and install the app.
4. Open the app and sign in with your Databricks credentials.

1. On your Android device, open the Google Play Store.
2. Search for **Databricks Genie**.
3. Download and install the app.
4. Open the app and sign in with your Databricks credentials.

## MCP connections

[MCP connections](/aws/en/genie-one/external-sources) configured in the Genie One web experience are available in the mobile app. You cannot add or configure MCP connections from the mobile app, but any connections already set up on the web are inherited automatically.

## Limitations

* Some dashboard functionality and visualization interaction might be limited on small-format displays.
* Apps might not render correctly if the app creator did not optimize them for mobile viewing. Build apps using the [Databricks AppKit](https://developers.databricks.com/docs/appkit/v0/) to optimize for rendering on mobile.
* Available in all [App Store countries](https://www.apple.com/choose-country-region/) except Belarus, China mainland, Russia, and Ukraine.
* Available in all [Google Play countries](https://support.google.com/googleplay/answer/143779) except Belarus, China, Cuba, Iran, and Russia.
