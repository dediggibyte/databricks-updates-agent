The Databricks Excel Add-in connects your Databricks workspace to Microsoft Excel, bringing governed Lakehouse data directly into your spreadsheets to help you move from data to decisions faster.

You can browse and import Databricks tables through an interface where no SQL knowledge is required. While the add-in offers the flexibility to execute custom SQL queries, it is optional. The add-in is fully supported across Excel for the web and desktop versions for Windows and macOS.

## Get started with the Excel Add-in

To use the Excel Add-in, you must first install and connect to your Databricks workspace, and then import and query data in Excel.

| Task | Description |
| --- | --- |
| [Set up the Excel Add-in](/aws/en/integrations/excel-setup) | Set up the Excel Add-in and connect to your Databricks workspace. You must complete this setup before you can query Databricks data in Excel. |
| [Import and query data in Excel](/aws/en/integrations/excel-query) | Import data from Databricks tables, write SQL queries, create pivot tables, and use custom functions to analyze your data in Excel. |
| [Write data back to Databricks](/aws/en/integrations/excel-write-back) | Write data from Excel to a Unity Catalog table. |
