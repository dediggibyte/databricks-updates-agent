Incremental refresh for materialized views in Databricks SQL

Databricks SQL materialized views now support incremental refresh in Public Preview.

Public Preview docs ↗
What it does

Databricks SQL materialized views now support incremental refresh in Public Preview. Instead of recomputing the entire result set on every refresh, the engine detects changes in base tables and updates only the affected rows, dramatically reducing refresh cost and latency for large aggregations.

Why it matters
